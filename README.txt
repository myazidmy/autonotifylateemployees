What do we have right now?
Script that can:
Send email if someone is late by 15mins
Send email if someone exceeds their break duration by 18mins
Send email if someone forgets to check in/out work and breaks

What data do we get?
Number of times someone exceeds their break per week
Number of times someone their break per month
Number of times someone late per week
Number of times someone late per month
Number of times someone forgets to use the tracker
With this data we can create a automated monthly/weekly report on the status of agents
